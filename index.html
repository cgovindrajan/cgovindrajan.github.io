BI ENRICHER FOR APACHE PIG

Open Source Project

I have designed and implement generic enricher framework for Apache Pig using Groovy. The framework provides the facility to programmers to write complex business transformation of data in simple groovy closures of framework without having to write PIG UDFS and spending extra efforts. Enricher has in build schema validation to avoid run time failure. It includes huge set of new features which are not part of standard PIG UDFS libraries.



MUTABLE TABLES FOR HIVE

Open Source Project

In this Project, I have designed a new adaptive schema based tables on top of hive framework which provides the facilities of custom fractural and dimensional rollups schemes for huge data volume without impacting existing partitions. All possible rollups are stored attributes into recal tables and then attributes are used to read specific data from partition for recalculation without changing existing partition. Performance is gained via regions scans in hive.In heavy testing, around 500 TB of data are recalculated within 10 mintues on standard cluster.
