MUTABLE TABLES FOR HIVE

Open Source Project

In this Project, I have designed a new adaptive schema based tables on top hive framework which provides the facilities of custom fractural and dimensional rollups schemes for huge data volume w/o impacting existing partitions. All possilbe rollups are stored attributes into recal tables and then attributes are used to read specific data from partition for recalculation without changing existing partition. Performance is gained via regions scans in hive.In heavy testing, around 500 TB of data are recalculated within 10 mintues on standard cluster.

SMART SECURITY AND SURVEILLANCE USING FACE RECOGNITION

Senior Project, May 2014

We designed and implemented a system that maintains a database of faces that are to be tracked, which could also be criminal mugshots, trains itself using these images and recognizes faces that are captured through a CCTV camera. We used the Baum Welch Algorithm and Hidden Markov Model for face recognition. Viola Jones Algorithm was used for detecting faces in videa frames. We observed an accuracy rate of about 87% after the implementation. The project was implemented in MATLAB.
